WireGuard 完整安装指南

本指南将引导您在 Linux (Ubuntu/Debian) 服务器上完成 WireGuard 的安装和配置。

步骤 1：安装 WireGuard

首先，更新您的包列表并安装 WireGuard。

sudo apt update
sudo apt install wireguard -y

步骤 2：启用 IP 转发

为了让服务器能将客户端的流量转发到互联网，必须启用 IP 转发。

1. 编辑 /etc/sysctl.conf 文件：

sudo nano /etc/sysctl.conf

2. 找到（或添加）以下行，并确保它**没有**被 # 注释掉：

net.ipv4.ip_forward = 1

3. 保存并关闭文件。然后应用更改使其立即生效：

sudo sysctl -p

步骤 3：配置防火墙 (UFW)

您必须允许流量进入您选择的 WireGuard 监听端口（默认为 51820/udp）。

sudo ufw allow 51820/udp
sudo ufw allow ssh

如果您尚未启用 UFW，请启用它：

sudo ufw enable

重要： 如果您使用的是云服务器（阿里云、腾讯云、AWS等），您**必须**同时在云控制台的“安全组”或“防火墙”规则中，放行 UDP:51820 端口。

步骤 4：使用配置管理器生成配置

这是最关键的一步。我们将使用本页面内置的工具来生成所有密钥和配置文件。

  1. 点击下方的“打开配置管理器”按钮。
  2. 在弹出的窗口中，填写“服务器公网 IP”。
  3. 检查“服务器网卡”是否正确。您可以在服务器上运行 ip route | grep default 来查找它（通常是 eth0 或 ens3）。
  4. 点击“**初始化 / 保存设置**”按钮。
  5. 点击“**+ 添加新客户端**”按钮来创建您需要的客户端。

步骤 5：应用服务器配置

1. 在管理器中，找到“2. 服务器配置 (wg0.conf)”框，点击“**复制配置**”按钮。

2. 在您的**服务器**上，创建并编辑 /etc/wireguard/wg0.conf 文件：

sudo nano /etc/wireguard/wg0.conf

3. 将您刚刚复制的配置**粘贴**进去，然后保存并关闭文件。

步骤 6：启动 WireGuard 服务

1. 启动服务：

sudo wg-quick up wg0

2. 设置开机自启：

sudo systemctl enable wg-quick@wg0

步骤 7：检查服务状态

运行 wg show 命令查看服务是否正常运行，以及客户端是否已连接。

sudo wg show

如果您看到 listening port: 51820 和已配置的 [Peer] 信息，则服务器已配置成功。

步骤 8：配置客户端

返回配置管理器，在“3. 客户端管理”中：

  • **对于电脑客户端：** 点击“**下载配置**”按钮，将 .conf 文件导入到您电脑的 WireGuard 客户端中。
  • **对于手机客户端：** 点击“**生成二维码**”按钮，使用手机 WireGuard App 扫描弹出的二维码。

连接后，再次在服务器上运行 sudo wg show，您应该能看到 latest handshake（最近握手）时间。

至此，全部配置完成！
